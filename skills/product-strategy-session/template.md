# Product Strategy Session Template

Use this template to capture the outputs of a strategy session.

## Template
```markdown
# Product Strategy Session Summary

## Strategy Context
- [Company goals / OKRs]
- [Market context / competition]

## Target Customer
- [Primary persona]
- [Key JTBD / needs]

## Problem Framing
- [Problem statement]
- [Why now]

## Opportunities
- [Top opportunities]

## Solution Hypotheses
- [Epic hypotheses]

## Prioritization
- [Framework used + top priorities]

## Roadmap / Sequencing
- [Now / Next / Later or quarters]

## Success Metrics
- [Key metrics]

## Next Steps
- [Experiments, discovery, PRD, roadmap planning]
```
