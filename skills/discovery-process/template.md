# Discovery Process Summary Template

Use this template to capture discovery outputs and decisions.

## Template
```markdown
# Discovery Summary

## Problem Framing
- [Problem statement]
- [Why now]

## Research Insights
- [Top customer pains]
- [Quotes / evidence]
- [Behavioral patterns]

## Personas / JTBD (Optional)
- [Primary persona]
- [Key jobs]

## Opportunities
- [Opportunity 1]
- [Opportunity 2]
- [Opportunity 3]

## Solution Hypotheses
- [Hypothesis 1]
- [Hypothesis 2]

## Experiments Run
- [Experiment + result]

## Decision
- [Go / No-Go / Pivot]

## Next Steps
- [PRD, roadmap, additional discovery]
```
